快捷键

  1. 简单快捷键

    • Ctrl+X 删除行
    • Ctrl+D 复制行
    • Ctrl+Shift+上键 方法上移
    • Ctrl+Q 显示注释文档
    • Ctrl+R 替换文本
    • Ctrl+F 查找文本
    • Alt+回车 导入包,自动修正
    • Alt+Insert generate方法
    • Ctrl+Shift+V 从剪贴板粘贴
    • ctrl+alt+v 自动补全函数返回值

  2. 使用到的记录

    1. 弹出类结构
      Ctrl+F12
    2. vue文件支持
      安装插件后 setting->file types-> 下面+号添加*.vue
    3. 查看类的继承关系
      Ctrl+H
    4. 快速打印变量
      soutv
    5. 抽取方法
      Ctrl+shift+m
    6. 批量修改变量名
      shift F6 或者 点击变量->右键->Refactor->Rename
    7. 大小写切换快捷键
      ctr+shift+u
    8. 快速搜索
      double shift 搜索所有
    9. 查找类或方法在哪被使用
      Alt+F7
    10. 查看子类方法实现
      Ctrl+Alt+B
    11. 查看接口的实现类
      格 ctrl + alt +B 或者 Ctrl+Alt+鼠标左键
    12. idea的debug基本用法&快捷键
      Alt + F8：计算表达式
      idea

  3. 查看API快捷键

    • ctrl+Q
    • ctrl+P

  4. 代码自动补全
    修改方法如下：
    点击 文件菜单(File) –> 点击 设置(Settings… Ctrl+Alt+S), –> 打开设置对话框。
    在左侧的导航框中点击 KeyMap。
    接着在右边的树型框中选择 Main menu –> Code –> Completion.
    接着需要做两件事：

    1. 移除原来的Cycle Expand Word 的 Alt+/ 快捷键绑定。
    2. 在 Basic 上点击右键,去除原来的 Ctrl+空格 绑定,然后添加 Alt+/ 快捷键。
      然后应用(Apply), OK.

功能

  • 添加模板
    • setting->live template->创建自定义的模板组, 在里面创建自己的模板, 选择作用范围
  • 修改创建的文件的模板, servlet&filter….
    • setting->file and code templates->other->web->修改

  • 使文件夹不参与编译

    • 右键->make Directory as ->Exclude

  • JSON

    字符串中写json: Alt+Enter 选择 injecti language of reference 选择JSON 再次A+E 开始Edit
